# MVPLoader
一款轻量级的MVP框架，只有少数几个Java文件，主要针对MVP架构中，内存泄露问题！

## 添加依赖
```Java
Step 1:

      allprojects {
          repositories {
              ...
              maven { url 'https://jitpack.io' }
          }
      }
      
Step 2:

      dependencies {
          compile 'com.github.albert-lii:MVPLoader:1.0.1'
      }
```
